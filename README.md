# Webcam Surveyor

## Overview

Webcam Surveyor is a software application designed for capturing and recording video from webcams on Windows systems. It serves purposes such as video surveillance, motion detection, and time-lapse recording, providing users with a straightforward interface to manage webcam footage.

## Key Features

**Video capture from connected webcams**  
**Motion detection with adjustable sensitivity**  
**Scheduled recording capabilities**  
**Time-lapse video creation**  
**Snapshot capture during video recording**  
**Configurable storage options for recorded files**  
**User-friendly interface optimized for Windows**  
**Support for multiple webcam devices**

## Why Webcam Surveyor?

Webcam Surveyor is developed with a focus on stable performance and clear video recording. It offers practical controls and settings that enhance usability for users needing reliable video capture on Windows platforms. The software emphasizes straightforward operation and consistent results for everyday video surveillance and recording needs.

## Benefits

Improved monitoring through motion-triggered recording
